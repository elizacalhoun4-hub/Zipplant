# ZipPlant Calendar 🌱

Your personal, smart planting calendar.  
Enter your ZIP code → get accurate indoor start dates, outdoor transplant windows, and recommended planting days based on real local weather and frost risk.

### Features
- Search any plant by name
- Filter: All | Vegetables | Ornamentals
- Personalized recommendations for 150+ popular annuals (tomatoes, zinnias, basil, sunflowers, etc.)
- Weather-aware risk indicators (green/yellow/red)
- Save your favorite plants
- Export a printable PDF calendar
- Works great on phones — add to home screen as an app

### How to Use
1. Enter your ZIP code
2. Browse or search plants
3. Tap any plant for exact timing
4. Save the ones you want to grow this season

Built as a fun vibe-coding project — completely free and open.

### Tech
- Streamlit (Python)
- Open-Meteo weather API (free, no key)
- Real USDA-style frost/planting logic

Made with ❤️ for gardeners who hate guessing when to plant.
